# SISTEN Design System

## Overview
Visual system for the Torres Eólicas do Nordeste (TEN) corporate portal. It aims for a professional, reliable, and high-quality aesthetic, using eolian energy-themed elements, solid blue brand colors, and high accessibility.

## Colors
- **Primary Blue (`#0056c6`):** The brand color, symbolizing wind, energy, and corporate reliability.
- **Neutral Light BG (`#f1f5f9`):** Slate-100 base for light theme.
- **Neutral Dark BG (`#0f172a`):** Slate-900 base for dark theme.

## Shapes
- **Large Border Radii:** Cards use 24px (`rounded-3xl`) and inputs/buttons use 12px (`rounded-xl`) to feel modern, soft, and premium.

## Do's and Don'ts
- **Do** respect the dark mode styles using tailwind's `dark:` classes.
- **Do** align features horizontally on large screens and stack on small screens.
- **Don't** use purple colors in buttons or controls (Brand guidelines).
